## Getting started

```
npm i encrypt-aes-js
```

or

```
yarn add encrypt-aes-js
```

## How to use

```
import { encryptAES } from "encrypt-aes-js";

const encryptData = encryptAES(secretKey, data);

```

## Attribute

| Attribute | type   | options |
| --------- | ------ | ------- |
| secretKey | string |         |
| data      | string |         |
